OpenCV Object Detection and Recognition
==================================================================

Prerequisite(s):
----------------

- python 3.5>
- pip
- numpy
- opencv
- opencv contribute
- imutils
- scipy
- PIL

.. _Prerequisites Installation Guide: https://github.com/ammar-khan/raspberry-pi-3-model-b-plus

Introduction:
-------------
This is an example of OpenCV object detection and recognition on Raspberry Pi 3 Model B+.

Configuration:
--------------
Refer to:

``src/common/package/config``

To Recognise/Stream:
